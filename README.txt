Pre-Requisite:
1. Install the most stable version of jdk ( version 11 or above).
2. Install the eclipse latest version.
3. Maven installed and configured in local and system variables.
4. In Eclipse Marketplace install TestNG plugin
5. Chrome browser installed.
6. Firefox and Egde browser can be installed as optional


Step to execute:
-> Import the A4Assessment file into Eclipse workspace
-> Find run_test.bat file in the Project Explorer
-> And then right-click the file and Run.
 	Note: If Run option not available, click Run Configuration and click Run

Post Report:
-> Find the extendReport.html File
-> Open with Web Browser
